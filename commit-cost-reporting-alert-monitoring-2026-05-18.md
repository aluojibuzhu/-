# Commit: 成本填报与预警监控迭代

## 范围

本次提交覆盖成本填报、成本审批、预警监控相关的新功能迭代，以及开发过程中发现的小范围缺陷修复。

## 主要改动

- 预警看板支持项目健康度、预算执行、成本趋势、成本科目占比展示。
- 预警趋势图支持按项目、年月日维度、一级成本科目切换。
- 报销申请的成本科目改为一级科目必填、二级科目可选，并与基础配置同步。
- 工时填报的工时类型改为与人工费二级成本科目同步，避免重复字段。
- 成本审批列表修复筛选自动刷新、成本科目展示一级科目等问题。
- 工时列表状态标签颜色调整，避免状态颜色重复并保持语义一致。

## 数据与脚本

- 新增报销成本科目可选二级科目的数据库兼容脚本。
- 调整预警监控相关表结构脚本，补充看板和规则所需字段。

## 验证

- `mvn -pl ruoyi-admin -am -DskipTests compile`
- `mvn -pl ruoyi-admin -am -DskipTests package`
- `npm run build:prod`
- 接口自测：工时类型与人工费二级成本科目同步。
- 接口自测：预警趋势图总计/科目维度切换。
