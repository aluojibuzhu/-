# 模块五成本看板提交说明

## 提交主题

实现成本看板与专项成本报表模块。

## 主要内容

- 新增成本总览看板，支持按项目状态查看合同金额、预算、已执行成本、余额、利润空间和本月入账。
- 新增四张专项报表页面：项目资金汇总表、科目明细表、入账流水明细表、节点执行表。
- 新增报表筛选与导出能力，筛选区统一为若依常用横向表单样式，字段名与输入框同一行。
- 打通报表后端数据接口，支持项目、成本科目、WBS节点、单据类型、审批人、入账时间等必要筛选。
- 新增模块五菜单初始化 SQL。

## 自测结果

- `npm run build:prod` 通过。
- 四张专项报表列表接口均返回 `200`。
- 入账流水按成本科目筛选正常。
- 入账流水 Excel 导出正常生成。
