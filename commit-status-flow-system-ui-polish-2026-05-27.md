# 提交说明：项目状态流转与系统界面美化

## 提交范围

- 完善项目状态按计划日期自动流转逻辑。
- 调整工时、报销在已完工项目下的填报规则。
- 重构登录页、首页工作台、个人中心和顶部栏文案。
- 更新系统名称与页脚版权，弱化默认若依模板痕迹。
- 增加业务日期修复 SQL，便于修正演示数据中的日期一致性问题。

## 关键改动

- 项目列表、项目详情和成本看板查询前会刷新项目状态：
  - 到达预计开工日期后进入进行中。
  - 到达预计竣工日期后进入已完工。
- 项目审批通过时会根据当前日期直接落到合理状态。
- 已完工项目允许继续填报成本。
- 工时填报日期必须在项目计划周期内。
- 报销申请不限制发生日期。
- 登录页改为极简风格，去除冗余说明文案。
- 首页改为简洁模块入口、公告板和运行概览。
- 个人中心修复乱码并统一为系统风格表单。
